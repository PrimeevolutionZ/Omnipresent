# Omnipresent
## Что это?
Программа с простым интерфейсом для загрузки нужных видео/звуков/музыки, что нацелена на нормисов как я.
<img width="1434" height="471" alt="Omnipresent 2" src="https://github.com/user-attachments/assets/8230a8d0-025c-4a28-a761-6542a41f3b5e" />
Поддерживается множество функций, которые реализованы при помощи утилит [yt-dlp](https://github.com/yt-dlp/yt-dlp) и [ffmpeg](https://ffmpeg.org/download.html).
